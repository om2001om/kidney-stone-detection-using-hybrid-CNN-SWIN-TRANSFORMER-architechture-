🔍 Squeeze-Excitation (SE) Block

Channel-wise Attention Mechanism for Deep CNNs

Overview: The Squeeze-Excitation block adaptively recalibrates channel-wise feature responses by explicitly modeling interdependencies between channels.

Key Innovation: Instead of treating all channels equally, SE blocks learn to emphasize informative features and suppress less useful ones.

3D Visualization: This view shows how the SE block transforms the feature tensor through spatial compression and channel recalibration.

Visual Metaphor: Think of it as a spotlight that learns which "channels" (feature detectors) to brighten or dim.

Mathematical Formulation: SE block can be described through a series of transformations on the input feature map.

🎓 Theoretical Insights

  • Information Bottleneck: The reduction ratio r creates an information bottleneck, forcing the network to learn compact channel relationships
  • Non-linear Modeling: The two FC layers with ReLU can model complex, non-linear channel dependencies
  • Sigmoid Gating: Unlike softmax (which creates competition), sigmoid allows multiple channels to be emphasized simultaneously
  • Differentiable: The entire SE block is differentiable, enabling end-to-end training with backpropagation
  • Scale Invariant: Global average pooling makes the squeeze operation invariant to input spatial resolution

import torch
import torch.nn as nn

class SEBlock(nn.Module):
    """
    Squeeze-and-Excitation Block
    
    Args:
        channel (int): Number of input channels
        reduction (int): Reduction ratio for bottleneck (default: 16)
    
    Shape:
        - Input: (B, C, H, W)
        - Output: (B, C, H, W)
    """
    def __init__(self, channel, reduction=16):
        super(SEBlock, self).__init__()
        
        # Squeeze: Global Average Pooling
        self.avg_pool = nn.AdaptiveAvgPool2d(1)
        
        # Excitation: Two FC layers with bottleneck
        self.fc = nn.Sequential(
            nn.Linear(channel, channel // reduction, bias=False),
            nn.ReLU(inplace=True),
            nn.Linear(channel // reduction, channel, bias=False),
            nn.Sigmoid()
        )
    
    def forward(self, x):
        # x shape: (batch_size, channels, height, width)
        b, c, h, w = x.size()
        
        # Squeeze: (B, C, H, W) -> (B, C, 1, 1)
        y = self.avg_pool(x)
        
        # Flatten: (B, C, 1, 1) -> (B, C)
        y = y.view(b, c)
        
        # Excitation: (B, C) -> (B, C)
        y = self.fc(y)
        
        # Reshape for broadcasting: (B, C) -> (B, C, 1, 1)
        y = y.view(b, c, 1, 1)
        
        # Scale: element-wise multiplication with broadcasting
        # (B, C, H, W) * (B, C, 1, 1) -> (B, C, H, W)
        return x * y.expand_as(x)


# Example usage
if __name__ == "__main__":
    # Create SE block for 512 channels
    se = SEBlock(channel=512, reduction=16)
    
    # Input tensor: batch_size=8, channels=512, height=14, width=14
    x = torch.randn(8, 512, 14, 14)
    
    # Forward pass
    output = se(x)
    
    print(f"Input shape: {x.shape}")
    print(f"Output shape: {output.shape}")
    print(f"Parameters: {sum(p.numel() for p in se.parameters()):,}")

    # Output shape: torch.Size([8, 512, 14, 14])

💻 Implementation Notes

  • AdaptiveAvgPool2d(1): Automatically handles any input spatial size, outputting 1×1 per channel
  • inplace=True: ReLU modifies tensors in-place to save memory during training
  • bias=False: Linear layers don't need bias when followed by BatchNorm (in full model)
  • expand_as(): Efficiently broadcasts the channel weights across spatial dimensions
  • Parameter Count: For C=512, r=16: 2×(512×32) = 32,768 parameters (very lightweight!)

class CNNBlockWithSE(nn.Module):
    """CNN Block with Squeeze-Excitation as used in your architecture"""
    def __init__(self, in_channels, out_channels, stride=2):
        super().__init__()
        
        self.conv1 = nn.Conv2d(in_channels, out_channels, 
                               kernel_size=3, stride=stride, padding=1)
        self.bn1 = nn.BatchNorm2d(out_channels)
        self.relu1 = nn.ReLU(inplace=True)
        self.se1 = SEBlock(out_channels, reduction=16)
        
        self.conv2 = nn.Conv2d(out_channels, out_channels, 
                               kernel_size=3, padding=1)
        self.bn2 = nn.BatchNorm2d(out_channels)
        self.relu2 = nn.ReLU(inplace=True)
        self.se2 = SEBlock(out_channels, reduction=16)
    
    def forward(self, x):
        # First conv block with SE
        out = self.conv1(x)
        out = self.bn1(out)
        out = self.relu1(out)
        out = self.se1(out)  # SE recalibration
        
        # Second conv block with SE
        out = self.conv2(out)
        out = self.bn2(out)
        out = self.relu2(out)
        out = self.se2(out)  # SE recalibration
        
        return out
